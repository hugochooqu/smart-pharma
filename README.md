# SmartPharma

SmartPharma is an AI-powered mobile application that provides
personalized herbal medicine recommendations by combining
traditional remedies with modern pharmacology and AI.

## Key Features
- User authentication (Expo + Appwrite)
- Symptom input and analysis using AI (Gemini)
- Personalized herbal recommendations
- Dosage reminders and progress tracking
- Clean, accessible mobile UI

## Tech Stack
- React Native (Expo)
- TypeScript
- Appwrite (Auth + Database)
- NativeWind
- Gemini AI

## Architecture Overview
SmartPharma uses a modular architecture with:
- Client-side state management
- Backend services for authentication and data persistence
- AI layer for symptom analysis and recommendations
